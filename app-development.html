Role: Lead developer • Tech: Kotlin, Room, WorkManager, Firebase (optional)

Overview: TaskMaster is a lightweight task manager with local-first sync, background reminders, and offline support. I designed the data model, implemented persistence using Room, and added scheduled notifications using WorkManager.

Outcome: Improved user engagement through reliable reminders and a small binary size optimized for older devices.
